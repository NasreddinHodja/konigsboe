** Example 1
Here, take this path:
#+id: 7
#+caption: R -> C -> H -> L -> C -> H
#+image: /images/blog/about-the-bridges/ex-01.png

Let's forget the rest of the lands and only look at H because thats the land whose bridges we managed to cross. Maybe we can find out why we could do it for H and not the rest?

We notice a few things:
1. H has 3 bridges
2. all of H's bridges were used! yay!
3. path started outside H and ended inside H (stuck)

I mean... it makes sense...

If H has 3 bridges, and the path started outside H, and we need to use all 3 brigdes once... then what happens is:

#+id: 8
#+caption: ... -> H -> ... -> H
#+image: /images/blog/about-the-bridges/h-01.png

1. one brigde will be used to enter H
2. a second one will be used to exit H
3. and the third one will be used (we must use all bridges), *ending the path in H*

DO YOU UNDERSTAND? WE HAVE SOMETHING HERE!!!!!!

*** Conjecture 1
:PROPERTIES:
:CUSTOM_ID: conjecture-1
:END:

I think I can confidently say that:
a) if a land has 3 brigdes,
b) and we want to use all of them,
c) startig outside,
THEN the path will end inside!

*** Conjecture 2
:PROPERTIES:
:CUSTOM_ID: conjecture-2
:END:

a) if a land has 3 brigdes,
b) and we want to use all of them,
c) we have 2 options:
   - if starting outside, [[#conjecture-1][THEN the path will end inside!]]
   - if startig inside, [[fig:10][THEN the path will end outside!]]

*** Conjecture 2 - reworded
:PROPERTIES:
:CUSTOM_ID: conjecture-3
:END:

In other words:
a) if a land has 3 bridges
b) and we have a path that crosses all bridges once
c) *then this land will have to be either at the start or at the end of the path*

* We have something already
Suppose such path is possible for our city and we found it.

1. In our [[fig:4][drawing]], we have 4 islands. 3 of them (R, L, and H) have 3 bridges.
2. [[#conjecture-3][We saw]] that if a land has 3 brigdes, it has to be either at the start or at the end of the path
3. This path will have 1 start land and 1 end land, naturally (it's a path, that's how they work usually... unless it's a cycle.......)

We have to fit 3 lands in 2 spots? R, L, and H all have to be either start land or end land.
